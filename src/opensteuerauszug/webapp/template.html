OpenSteuerAuszug

GitHub

Open-source and free service to generate Swiss e-tax statements (eSteuerauszug) from IBKR Flex Query XML reports.

Note: Although this online version produces the same output as the locally installed version, it is intended primarily for testing and demonstration. The installed offline version is clearly the suggested way to generate real PDFs for privacy and security reasons.

Frequently Asked Questions

Is this service free?

Yes, this service is completely free and without any guarantee. You are fully responsible for verifying the correctness of the generated PDF.

How can I report issues?

This project is fully open source. Reports are very welcome to improve the software. Please report issues at GitHub Issues.

Are there alternatives?

Yes, there are paid alternatives available:

  • ibkr.datalevel.ch
  • etaxify.ch
Where can I find the source code?

The full source code is available at GitHub under open-source MIT licence.

Do you store or track the uploaded data?

No uploaded data is stored longer than necessary for processing or tracked in this online version.

Where is the online service hosted?

The online service is hosted at Render.com free plan ("Hobby"). This may cause a longer initial startup time after a period of inactivity.
