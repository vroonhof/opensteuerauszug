{: .long-version }

Ceci n'est pas un extrait fiscal officiel. Il ne contient que des codes-barres
pour importer les données bancaires dans le logiciel fiscal. Il est destiné
exclusivement à l'import dans des logiciels de préparation fiscale tels que "PrivateTax" qui peuvent corriger les calculs fiscaux.

- Veuillez vérifier attentivement les informations de cet extrait.
- Vous pouvez importer cet extrait dans votre logiciel fiscal.
- Pour plus d'informations, voir [docs/user_guide.md](https://github.com/vroonhof/opensteuerauszug/blob/main/docs/user_guide.md).

### Actions importantes à effectuer

- **Vérifier les données** : Vérifiez qu'aucune erreur ou avertissement n'a été généré pendant le processus.
- **Titres** : Assurez-vous que tous vos titres sont listés et correctement associés au bon numéro de valeur.
- **Transactions** : Confirmez que toutes les transactions de l'année fiscale sont présentes et correctes et que les totaux semblent plausibles.
- **Positions** : Vérifiez que les positions de fin d'année pour tous vos avoirs sont exactes.
- **Liquidités** : Vérifiez que tous les soldes de liquidités sont corrects.
- **Revenus** : Vérifiez que tous les revenus d'intérêts et de dividendes sont correctement déclarés.
- **Recalculer dans le logiciel fiscal** : Après l'import de ce document dans votre logiciel fiscal, utilisez sa fonction pour **recalculer toutes les valeurs fiscales** sur la base des données officielles de la Kursliste.
- **Vérifier les données importées dans le logiciel fiscal** : PrivateTax permet par exemple d'afficher les informations structurées importées dans la vue d'édition.
