## Purpose

This is not an official tax statement. It was created by the
taxpayer(s) using OpenSteuerauszug (https://github.com/vroonhof/opensteuerauszug) from electronic
data of the financial institution. It only contains barcodes so that the
bank data can be imported into the tax software.

## Data

Responsibility for accuracy and completeness still lies with the taxpayer.

- He/she checks holdings, transactions, Valor numbers and interest payments.
- *No tax calculations were performed.* Where required by the format, values and totals are set to zero.
- The tax calculation based on the Kursliste should be taken over by the tax software and/or assessment software.

## Why a tax statement?

Unfortunately there is no standardized way to import only position or
transaction data into PrivateTax or similar software.

Foreign financial institutions do not provide (e-)tax statements. The
author of the OpenSteuerauszug software believes that automated data
transfer is still beneficial, especially since tax software is able to
correct and/or complete the tax-relevant calculations.
